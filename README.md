# bastard-pixel-battale
## Description
Bastard brother of pixel battle without delay, but with bastard net code. 

## Server
### hardcoded params
look server's main 

`PORT` - port to run server on
### setup

```bash
mkdir ./server/build
cd server/build
cmake ..
make
sudo ./BastardPixelBattle 
```
Server running

## Client
### hardcoded params
look client's main 

`DOMAIN` - server's ip or domain 

`PORT` - server's port
### setup
```bash
mkdir ./client/build
cd server/build
cmake ..
make
sudo ./client 
```
client running
